📌 Project Overview

The AI-Powered Campus Placement Intelligence System is an end-to-end machine learning and analytics platform designed to predict student placement outcomes based on academic performance, skill indicators, and practical exposure.

This system simulates real-world campus placement data, applies machine learning models to analyze student profiles, and provides interactive dashboard insights for data-driven decision-making.

The project demonstrates the complete ML pipeline — from data generation to model deployment.

🎯 Problem Statement

Campus placement decisions are often based on academic scores and interviews, but there is no structured data-driven system to evaluate student placement readiness.

This project aims to:

Predict placement likelihood using machine learning

Estimate potential salary for placed students

Analyze placement trends through visual dashboards

Provide data-driven insights for institutions

🧠 Machine Learning Approach
1️⃣ Data Generation

A synthetic dataset of 500 students was created including:

CGPA

Aptitude Score

Technical Test Score

Internships

Projects

Communication Skills

A weighted scoring mechanism determines placement status.

2️⃣ Classification Model

A Random Forest Classifier is used to:

Predict placement status (Placed / Not Placed)

Generate placement probability score

3️⃣ Regression Model

A Random Forest Regressor predicts:

Expected salary (in LPA) for placed students

4️⃣ Interactive Dashboard

Built using Streamlit, the dashboard allows:

Real-time student input analysis

Placement probability display

Salary prediction

Feature importance visualization

Placement distribution charts

Dataset preview
